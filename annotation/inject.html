# @Inject

@Inject和@Autowride的使用规则是一样的，只是不能设置require属性

# 使用规则

1. 默认按照类型去容器中找对应的组件，如果找到了就自动装配上去(只有一个这种类型组件的情况)

2. 如果存在多个相同类型的组件，那么就会按照属性的名称作为id去容器中查找

3. 如果存在多个相同类型的组件，那么就可以使用`@Qualifier`注解指定需要配置Bean的id名称

4. 如果存在多个相同类型的组件，可以通过`@Primary`设置Bean的优先级别

# 代码实现

  1. 导入jar包
<!-- https://mvnrepository.com/artifact/javax.inject/javax.inject -->
<dependency>
    <groupId>javax.inject</groupId>
    <artifactId>javax.inject</artifactId>
    <version>1</version>
</dependency>
  1. 其他代码不变，与@Service注解一样，只需要修改BookService的内容
package com.hzj.service;

import com.hzj.dao.BookDao;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.beans.factory.annotation.Qualifier;
import org.springframework.stereotype.Service;

import javax.annotation.Resource;
import javax.inject.Inject;

/**
 * 使用@Service：默认是类名首字母小写bookService
 * @author hzj
 */
@Service
public class BookService {

    //通过@Qualifier指定Bean的id名称
    @Qualifier("bookDao2")
    @Inject
    public BookDao bookDao;


    public BookService() {
    }

    public BookDao getBookDao() {
        return bookDao;
    }

    public void setBookDao(BookDao bookDao) {
        this.bookDao = bookDao;
    }

    @Override
    public String toString() {
        return "BookService{" +
                "bookDao=" + bookDao +
                '}';
    }
}
