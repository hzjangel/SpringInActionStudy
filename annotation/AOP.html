# Spring AOP

# 步骤

  1. 使用@Aspect注解定义一个切面

  2. 在配置类使用@EnableAspectJAutoProxy开启基于注解的AOP模式

# 通知方法

  • 前置通知(@Before)：在目标方法运行之前

  • 后置通知(@After)：在目标方法运行之后

  • 返回通知(@AterReturn)：在目标方法正常返回之后

  • 异常通知(@AfterThrowing)：在目标方法出现异常之后运行

  • 环绕通知(@Around)：动态代理，手动推荐目标方法运行

# 代码实现

  1. 编写业务逻辑类
package com.hzj.aop;

/**
 * @author hzj
 */
public class MathCalulator {

    public double div(int num1, int num2) {
        System.out.println("方法运行了");
        return num1 / num2;
    }
}
  1. 编写业务逻辑类的切面

在定义切面的时候，需要使用@Aspect表明这是一个切面类，使用切入点表达式表明切入的地方，由于为每一个方法都写一个一样的切入点表达式，所以可以把切入点表达式使用@Pointcut注解将切入点表达式抽取出来

package com.hzj.aop;

import org.aspectj.lang.JoinPoint;
import org.aspectj.lang.annotation.*;

import java.lang.reflect.Array;
import java.util.ArrayList;
import java.util.Arrays;

/**
 * @author hzj
 */
@Aspect
public class LogAspects {

    /**
     * 抽取公共切入表达式
     * 1. 本类引用:直接写方法名就可以了
     * 2. 其他页面引用:需要加上路径名
     */
    @Pointcut("execution(public * com.hzj.aop.MathCalulator.*(..))")
    public void pointCut() {}

    /**
     * 使用相对路径
     * @param joinPoint
     */
    @Before("pointCut()")
    public void logStart(JoinPoint joinPoint) {

        Object[] args = joinPoint.getArgs();
        System.out.println(joinPoint.getSignature().getName() + "方法正常运行...参数列表是" + Arrays.asList(args));
    }

    /**
     * 使用绝对路径
     * @param joinPoint
     */
    @After("com.hzj.aop.LogAspects.pointCut()")
    public void logEnd(JoinPoint joinPoint) {
        System.out.println(joinPoint.getSignature().getName() + "方法正常结束...");

    }

    @AfterReturning(value = "pointCut()", returning = "result")
    public void logReturn(JoinPoint joinPoint, Object result) {
        System.out.println(joinPoint.getSignature().getName() + "方法正常结束...返回值是{"+ result +"}");
    }

    @AfterThrowing(value = "pointCut()", throwing = "exception")
    public void logException(JoinPoint joinPoint, Exception exception) {
        System.out.println(joinPoint.getSignature().getName() + "方法运行异常...异常是..." + exception);

    }

}
  1. 编写配置类，将业务逻辑类和业务逻辑的切面注入到IOC容器中

在配置类中需要加上@EnableAspectJAutoProxy开启基于注解的AOP模式

package com.hzj.config;

import com.hzj.aop.LogAspects;
import com.hzj.aop.MathCalulator;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.context.annotation.EnableAspectJAutoProxy;

/**
 * 导入aop模块
 *  定义一个业务逻辑类，在业务逻辑运行时候将日志进行打印
 *   通知方法
 *      前置通知(@Before)：在目标方法运行之前
 *      后置通知(@After)：在目标方法运行之后
 *      范湖通知(@AterReturn)：在目标方法正常返回之后
 *      异常通知(@AfterThrowing)：在目标方法出现异常之后运行
 *      环绕通知(@Around)：动态代理，手动推荐目标方法运行
 * @author hzj
 */
@Configuration
@EnableAspectJAutoProxy
public class MainConfigAop {

    @Bean
    public MathCalulator mathCalulator() {
        return new MathCalulator();
    }

    @Bean
    public LogAspects logAspects() {
        return new LogAspects();
    }
}
  1. 编写一个测试类
package com.hzj.test;

import com.hzj.aop.MathCalulator;
import com.hzj.config.MainConfigAop;
import org.junit.Test;
import org.springframework.context.annotation.AnnotationConfigApplicationContext;

public class AspectsTest {

    @Test
    public void test01() {
        AnnotationConfigApplicationContext applicationContext = new AnnotationConfigApplicationContext(MainConfigAop.class);

        MathCalulator mathCalulator = applicationContext.getBean(MathCalulator.class);
        mathCalulator.div(1, 1);
    }
}
