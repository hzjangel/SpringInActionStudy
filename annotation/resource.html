# @Resource

@Resource和@Autowired一样，也可以自动装配，但是在功能上面没有@Autowride那么强大

# 使用规则

  1. 默认按照类型去容器中找对应的组件，如果找到了就自动装配上去(只有一个这种类型组件的情况)

  2. 如果存在多个相同类型的组件，那么就会按照属性的名称作为id去容器中查找

  3. 可以使用@Resource的name属性指定Bean

@Resource只有这两个规则，无法与Qualifier和@Primary一起使用

# 代码实现

  1. 其他内容和@Autowride的内容一样

  2. 更改BookService的内容

package com.hzj.service;

import com.hzj.dao.BookDao;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.beans.factory.annotation.Qualifier;
import org.springframework.stereotype.Service;

import javax.annotation.Resource;

/**
 * 使用@Service：默认是类名首字母小写bookService
 * @author hzj
 */
@Service
public class BookService {

    //通过@Qualifier指定Bean的id名称
//    @Qualifier("bookDao")
//    @Autowired(required = false)
//    @Autowired
    //@Resource(name = "bookDao2")
    @Resource
    public BookDao bookDao;


    public BookService() {
    }

    public BookDao getBookDao() {
        return bookDao;
    }

    public void setBookDao(BookDao bookDao) {
        this.bookDao = bookDao;
    }

    @Override
    public String toString() {
        return "BookService{" +
                "bookDao=" + bookDao +
                '}';
    }
}
