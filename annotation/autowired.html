# @Autowired

@Autowired是自动装配，是Spring利用依赖注入(DI)，完成IOC容器中各个组件的依赖关系赋值

# 使用规则

  1. 默认按照类型去容器中找对应的组件，如果找到了就自动装配上去(只有一个这种类型组件的情况)

  2. 如果存在多个相同类型的组件，那么就会按照属性的名称作为id去容器中查找

  3. 如果存在多个相同类型的组件，那么就可以使用@Qualifier注解指定需要配置Bean的id名称

  4. 如果存在多个相同类型的组件，可以通过@Primary设置Bean的优先级别

  5. @Autowired默认是需要存在这个Bean的，如果不存在那么程序就会报错，可以通过设置required属性来确定这个Bean是否是必须要有的

  1. 新建一个Bean配置类
package com.hzj.config;

import com.hzj.dao.BookDao;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.ComponentScan;
import org.springframework.context.annotation.Configuration;
import org.springframework.context.annotation.Primary;

/**
 * @author hzj
 */
@Configuration
@ComponentScan(value = {"com.hzj.service", "com.hzj.dao"})
public class MainConfigOfAutowired {


    /**
     * 通过@Primary设置Bean的优先级别
     * @return
     */
    @Primary
    @Bean(value = "bookDao2")
    public BookDao bookDao() {
        return new BookDao();
    }
}

package com.hzj.service;

import com.hzj.dao.BookDao;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.beans.factory.annotation.Qualifier;
import org.springframework.stereotype.Service;

/**
 * 使用@Service：默认是类名首字母小写bookService
 * @author hzj
 */
@Service
public class BookService {

    //通过@Qualifier指定Bean的id名称
    @Qualifier("bookDao")
//    @Autowired(required = false)

    public BookDao bookDao;


    public BookService() {
    }

    public BookDao getBookDao() {
        return bookDao;
    }

    public void setBookDao(BookDao bookDao) {
        this.bookDao = bookDao;
    }

    @Override
    public String toString() {
        return "BookService{" +
                "bookDao=" + bookDao +
                '}';
    }
}

package com.hzj.test;

# 在参数上使用

不与@Bean一起使用，那么@Autowride注解只有在构造方法的参数上使用，并且有一定的限制

package com.hzj.bean;

import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Component;

/**
 * @author hzj
 */
@Component
public class Boss {

    private Car car;

    private Cat cat;



    public Boss() {
        System.out.println("调用了Boss的无参构造方法");
    }

    /**
     * 在构造方法上使用@Autowired注解
     * @param car
     * @param cat
     */
    public Boss(@Autowired Car car, @Autowired Cat cat) {
        System.out.println("调用了Boss的有参构造方法");
        this.car = car;
        this.cat = cat;
    }
    
    public Car getCar() {
        return car;
    }

    public void setCar(Car car) {
        System.out.println("调用了Boss的setCar方法");
        this.car = car;
    }

    public Cat getCat() {
        return cat;
    }

    public void setCat(Cat cat) {
        System.out.println("调用了Boss的setCat方法");
        this.cat = cat;
    }

    @Override
    public String toString() {
        return "Boss{" +
                "car=" + car +
                ", cat=" + cat +
                '}';
    }
}

运行测试类的时候发现调用的是Boss的无参构造方法，并没有调用个Boss的有参构造方法，所以并没有完成自动注入

# 与@Bean配合使用

在MainConfigOfAutowired类下使用@Bean

package com.hzj.config;

import com.hzj.bean.Boss;
import com.hzj.bean.Car;
import com.hzj.bean.Cat;
import com.hzj.dao.BookDao;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.ComponentScan;
import org.springframework.context.annotation.Configuration;
import org.springframework.context.annotation.Primary;

/**
 * @author hzj
 */
@Configuration
@ComponentScan(value = {"com.hzj.service", "com.hzj.dao", "com.hzj.bean"})
public class MainConfigOfAutowired {


    /**
     * 通过@Primary设置Bean的优先级别
     * @return
     */
    @Primary
    @Bean(value = "bookDao2")
    public BookDao bookDao() {
        return new BookDao("2");
    }

    @Bean("boss1")
    public Boss boss(@Autowired Car car,@Autowired Cat cat) {
        return new Boss(car, cat);
    }
}

与@Bean配合使用的时候，@Autowride也可以省略
